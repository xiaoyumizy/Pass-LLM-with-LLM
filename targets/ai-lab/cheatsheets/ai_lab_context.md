## 1. 实验室简介

- **全称**：上海人工智能实验室（Shanghai AI Laboratory），简称"上海AI实验室"/"浦江实验室"
- **成立**：2020年，上海市与上海交大、复旦、浙大等联合支持成立，新型研发机构
- **定位**：聚焦通用人工智能（AGI）前沿研究，兼顾AI基础研究与产业赋能
- **方向**：大语言模型、多模态模型、AI for Science、具身智能、可信AI、评测体系
- **特点**：强调开源开放，国内大模型开源生态核心推动者之一

## 2. 核心大模型项目

### InternLM（书生大语言模型）

- InternLM-7B/20B（初代）-> InternLM2（GQA、200K+上下文、强推理数学）-> InternLM3（待确认，强化CoT与代码）
- 核心特点：中文能力强、长上下文、数学推理、代码生成
- 全系列GitHub/HuggingFace开源，Apache 2.0可商用

### InternVL（书生视觉语言模型）

- InternVL 1.5 -> 2.0 -> 2.5 -> 3.0，动态分辨率输入，强OCR/文档理解
- 核心能力：图像描述、VQA、图表理解、多图推理、视频理解
- 架构：ViT-InternLM，视觉编码器与语言模型解耦，便于扩展
- 多模态榜单多次SOTA（MMBench、MMMU等，待确认时间）

### 其他项目

- **InternEvo**：高效预训练框架，3D混合并行（数据+张量+流水线）
- **XTuner**：轻量微调工具箱，支持SFT/RLHF（待确认）
- **LMDeploy**：推理部署引擎，TurboMind加速、量化、API Server
- **Lagent/AgentLego**：智能体框架，工具调用与多步推理

## 3. AI for Science 方向

### 布局

- 设有专门 **AI for Science 中心**，推动AI与物理/化学/气象/生命科学交叉
- 理念：用大模型+科学数据构建"科学基座模型"，加速科学发现

### 地球空间/气象/科学模拟

- **风乌（Fengwu）系列**：气象大模型，中期天气预报，全球分辨率0.25度，与华为盘古气象并列国内领先
- **风乌-GHR**：更高分辨率版本（待确认），支持台风路径等极端天气预测
- **科学模拟加速**：神经网络替代部分PDE求解器计算（流体模拟、气候模拟）
- **地球空间**：遥感影像理解、地理空间信息提取（待确认具体项目名）

### GNN/扩散模型在科学领域的结合

- GNN：分子性质预测、蛋白质结构建模、材料属性推断
- 扩散模型：分子构象生成、蛋白质设计、天气预报中的生成式方法（如扩散集合预报）
- 可能涉及蛋白质结构预测（对接AlphaFold范式）（待确认）

## 4. AGI 方向

- 愿景明确追求 **AGI**，路径：大语言模型为基座 -> 多模态 -> 具身智能 -> 工具使用 -> 自主推理
- 重视 **Scaling Law** 研究（模型规模、数据量与能力涌现关系）
- 关注安全可控：可信AI、对齐（Alignment）、红队测试
- AGI要素：强推理、多模态理解、世界模型、自主学习能力

## 5. 技术特色

- **开源全栈**：InternLM + InternVL + LMDeploy + XTuner + OpenCompass，GitHub组织：InternLM、OpenGVLab
- **OpenCompass（司南）**：国内主流大模型评测平台，50+数据集，涵盖知识/推理/代码/数学/中文/多模态
  - 提供CompassRank排行榜 + 标准化评测流水线，可扩展自定义评测
  - 同类：C-Eval、MMLU、HELM，OpenCompass定位一站式综合平台
- **InternEvo训练框架**：3D混合并行 + FlashAttention + 混合精度(bf16/fp16) + HuggingFace兼容
