## 一、GNN 图神经网络

### 1. 图的基本概念

| 概念 | 定义 | 速记 |
|------|------|------|
| 邻接矩阵 A | A_ij=1 表示节点 i 和 j 有边 | 描述"谁和谁相连" |
| 度矩阵 D | 对角阵，D_ii = 节点 i 的邻居数 | 描述"每个节点有几条边" |
| 拉普拉斯矩阵 L | L = D - A；对称归一化 L_sym = D^{-1/2} L D^{-1/2} | 谱域图卷积的基石，特征值反映图的频率 |

### 2. GCN (Graph Convolutional Network)

| 项目 | 内容 |
|------|------|
| 核心公式 | H^{(l+1)} = sigma( D_tilde^{-1/2} A_tilde D_tilde^{-1/2} H^{(l)} W^{(l)} )，其中 A_tilde = A + I |
| 聚合方式 | 对自身 + 所有邻居的特征取加权平均（权重由度归一化决定，**无学习权重**） |
| 一句话 | "把邻居特征做归一化平均后线性变换 + 非线性激活，逐层传播" |

### 3. GAT (Graph Attention Network)

| 项目 | 内容 |
|------|------|
| 注意力权重 | alpha_ij = softmax( LeakyReLU( a^T [Wh_i \|\| Wh_j] ) )，对每条边学习一个标量权重 |
| 与 GCN 区别 | GCN 邻居权重固定（由度决定）；GAT 用可学习注意力自适应分配邻居权重，多头注意力提升表达力 |

### 4. 消息传递框架 (Message Passing Neural Network, MPNN)

| 步骤 | 公式/描述 | 作用 |
|------|-----------|------|
| Message | m_ij = phi(h_i, h_j, e_ij) | 每条边生成消息 |
| Aggregate | M_i = AGG({m_ij : j in N(i)}) | 聚合邻居消息（sum/mean/max） |
| Update | h_i' = UPDATE(h_i, M_i) | 更新节点表示 |

> GCN/GAT/GraphSAGE/GIN 都是 MPNN 的特例。面试必答。

### 5. 任务层级

| 层级 | 典型任务 | 输出位置 |
|------|----------|----------|
| 节点级 | 节点分类、链接预测 | 节点嵌入 h_i |
| 边级 | 边分类、关系预测 | 拼接/乘积 h_i, h_j |
| 图级 | 图分类、图回归 | Readout（sum/mean/attention pooling） |

### 6. GNN 应用场景

| 领域 | 示例 |
|------|------|
| 分子性质预测 | 原子=节点，化学键=边；预测溶解度、毒性、生成力 |
| 交通流量 | 路口=节点，路段=边；预测拥堵/流量 |
| 地球空间 | 气象站点网络、大气/海洋网格数据建模、MeshGraphNets 求解 PDE |

## 二、扩散模型 (Diffusion Model)

### 1. 核心思想与 DDPM 训练

> **前向**逐步加噪直到纯高斯噪声 -> **反向**学习逐步去噪恢复数据。DDPM 训练目标：**预测噪声 epsilon**，L = ||epsilon - epsilon_theta(x_t, t)||^2。采样时用公式逐步还原。

### 2. 前向过程 (加噪) / 反向过程 (去噪)

| 项目 | 内容 |
|------|------|
| 前向 q(x_t\|x_{t-1}) | N(x_t; sqrt(1-beta_t) x_{t-1}, beta_t I)；一步到位：q(x_t\|x_0) = N(sqrt(alpha_bar_t) x_0, (1-alpha_bar_t) I) |
| 反向 p(x_{t-1}\|x_t) | N(x_{t-1}; mu_theta(x_t,t), sigma_t^2 I)，神经网络学习逐步去除噪声 |
| 速记 | 前向每步乘 sqrt(1-beta) 加噪；反向由 UNet 参数化去噪 |

### 3. 与 GAN / VAE 对比

| 模型 | 一句话 |
|------|--------|
| GAN | 生成器与判别器对抗训练，模式坍塌风险，训练不稳定 |
| VAE | 编码器-解码器，潜在空间正则化，生成质量偏模糊 |
| Diffusion | 逐步去噪，训练稳定（固定前向过程），生成质量高但采样慢 |

### 4. 条件生成：Guidance

| 方法 | 原理 | 优劣 |
|------|------|------|
| Classifier Guidance | 训练一个分类器，在采样时用梯度引导去噪方向 | 需要额外分类器，类别受限 |
| Classifier-Free Guidance (CFG) | 训练时随机丢弃条件，推理时用条件预测与无条件预测的线性外推：epsilon' = epsilon_uncond + s*(epsilon_cond - epsilon_uncond) | 无需额外模型，主流方案 |

### 5. Stable Diffusion / Latent Diffusion

| 项目 | 内容 |
|------|------|
| 核心改进 | 在 **潜空间** 而非像素空间做扩散：先用 VAE 编码器压缩图像到低维 latent，再在 latent 上做扩散去噪，最后解码回图像 |
| 优势 | 计算量大幅降低（如 512x512 -> 64x64 latent），可消费级 GPU 运行 |
| 条件注入 | 用 Cross-Attention 将文本条件（CLIP 编码）注入 UNet |

### 6. 扩散模型应用

| 领域 | 示例 |
|------|------|
| 图像/视频生成 | DALL-E、Sora（时空 diffusion transformer） |
| 科学模拟 | 分子构象生成、蛋白质结构采样 |
| 气象/地球科学 | GraphCast、Pangu-Weather 等用扩散做集合预报；降水临近预报 |

## 三、与 AI for Science 的关联

| 方向 | GNN 角色 | 扩散模型角色 |
|------|----------|--------------|
| 分子/药物设计 | 分子图上的性质预测、虚拟筛选 | 分子构象生成、3D 分子生成（DiffSBDD） |
| 材料科学 | 晶体图预测形成能/带隙 | 材料结构生成、晶体对称性采样 |
| 地球/大气科学 | 气象站点图网络、MeshGraphNets | 降水/极端天气的集合预报、不确定性量化 |
| 蛋白质 | 蛋白质接触图预测 | 蛋白质结构生成（RFDiffusion） |

> **面试串联点**：GNN 擅长处理**非欧几里得结构数据**（图/网格），扩散模型提供**高质量生成与不确定性建模**。两者结合是 AI4Science 前沿，如"GNN 做编码器 + 扩散做解码器"的范式正在分子设计和气候预测中兴起。
