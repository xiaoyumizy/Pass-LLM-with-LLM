## 一、线性代数

| 知识点 | 速记 | 常见考法 |
|--------|------|----------|
| 矩阵乘法 | (AB)^T = B^T A^T；维度 m×n 乘 n×p 得 m×p | 给 A、B 维度，判断哪个乘法可执行 |
| 转置 | (A^T)^T = A；对称矩阵 A = A^T | 判断矩阵是否对称/反对称 |
| 逆矩阵 | AA^{-1} = I；(AB)^{-1} = B^{-1}A^{-1}；满秩方阵可逆 | 判断哪些矩阵有逆 / 计算 2×2 逆 |
| 行列式 | det(AB) = det(A)det(B)；det(A^T) = det(A)；全零行则 det=0 | 判断 det 值是否为零（可逆性） |
| 秩 | rank(AB) ≤ min(rank(A), rank(B))；秩 = 列空间维数 = 行空间维数 | 求秩或判断方程组解的个数 |
| 特征值与特征向量 | Av = λv；对称矩阵特征值均为实数且特征向量正交 | 求 2×2 矩阵特征值 / 判断迹与特征值关系 |
| 迹 | tr(A) = 对角线之和 = 特征值之和；tr(AB) = tr(BA) | 用迹的性质快速判断特征值之和 |
| 正交矩阵 | Q^T Q = I；det(Q) = ±1；保长保角（旋转/反射） | 判断 Q^T Q 是否等于单位阵 |
| SVD 分解 | A = UΣV^T，U/V 正交，Σ 对角非负；σ_i = sqrt(λ_i(A^T A)) | 判断 SVD 各矩阵的维度或性质 |
| PCA 原理 | 对协方差矩阵做特征分解；取前 k 个最大特征值对应方向 | 判断 PCA 降维后的维数 / 方差贡献率计算 |

## 二、概率统计

| 知识点 | 速记 | 常见考法 |
|--------|------|----------|
| 条件概率 | P(A\|B) = P(AB) / P(B)；独立则 P(AB) = P(A)P(B) | 判断独立性 / 计算联合概率 |
| 贝叶斯公式 | P(A\|B) = P(B\|A)P(A) / P(B)；后验 ∝ 似然 × 先验 | 给出先验和似然，求后验概率 |
| 全概率公式 | P(B) = Σ P(B\|A_i)P(A_i)，要求 A_i 互斥且穷举 | 计算分母 P(B) 再代入贝叶斯 |
| 正态分布 | N(μ, σ²)；68-95-99.7 法则；标准化 Z=(X-μ)/σ ~ N(0,1) | 求某区间概率 / 判断 Z 值 |
| 均匀分布 | U(a,b)：期望=(a+b)/2，方差=(b-a)²/12 | 计算期望方差 / 区间概率 |
| 泊松分布 | P(X=k) = λ^k e^{-λ} / k!；期望=方差=λ | 稀有事件建模 / 判断期望与方差关系 |
| 伯努利 & 二项 | Bernoulli: P(1)=p；Binomial(n,p): E=np, Var=np(1-p) | 计算 n 次独立试验中成功 k 次的概率 |
| 期望性质 | E[aX+b] = aE[X]+b；独立时 E[XY] = E[X]E[Y] | 利用线性性快速算期望 |
| 方差性质 | Var(aX+b) = a²Var(X)；独立时 Var(X+Y) = Var(X)+Var(Y) | 判断 Var(X-Y) 是否等于 Var(X)+Var(Y) |
| 协方差与相关系数 | Cov(X,Y) = E[XY]-E[X]E[Y]；ρ = Cov/(σ_X σ_Y)，ρ ∈ [-1,1] | 判断不相关与独立的区别 |
| 大数定律 | 样本均值 → 期望（n→∞）；频率 → 概率 | 判断增加样本量对均值估计的影响 |
| 中心极限定理 | 独立同分布样本均值 → 正态分布（n 足够大） | 判断样本均值的近似分布 |
| MLE vs MAP | MLE: 最大似然；MAP: 最大后验 = MLE + 先验；MAP 正则化等价 | 判断 L2 正则化对应什么先验（高斯先验） |

## 三、微积分

| 知识点 | 速记 | 常见考法 |
|--------|------|----------|
| 偏导数 | ∂f/∂x_i：固定其他变量对 x_i 求导 | 求 f(x,y)=x²y+3y 的偏导 |
| 梯度 | ∇f = (∂f/∂x₁, ..., ∂f/∂x_n)；方向为最大增长方向 | 判断梯度方向 / 梯度为零的点 |
| 链式法则 | dz/dx = (dz/dy)(dy/dx)；多变量：∂z/∂x = Σ (∂z/∂y_i)(∂y_i/∂x) | 计算复合函数导数 / 理解反向传播 |
| Jacobian 矩阵 | J_{ij} = ∂f_i/∂x_j；f: R^n → R^m 则 J 为 m×n | 判断 Jacobian 维度 / 向量函数求导 |
| Hessian 矩阵 | H_{ij} = ∂²f/∂x_i∂x_j；对称矩阵；正定 → 极小值 | 判断驻点类型（极大/极小/鞍点） |
| 泰勒展开 | f(x) ≈ f(a) + f'(a)(x-a) + ½f''(a)(x-a)² + ... | 一阶/二阶近似判断函数值 |
| 凸函数性质 | f(λx+(1-λ)y) ≤ λf(x)+(1-λ)f(y)；局部最优=全局最优 | 判断函数是否凸 / 梯度下降收敛性 |

## 四、深度学习相关数学

| 知识点 | 速记 | 常见考法 |
|--------|------|----------|
| Softmax | softmax(z)_i = e^{z_i} / Σ e^{z_j}；输出和=1，平移不变 softmax(z+c)=softmax(z) | 判断数值稳定性（减最大值）/ 平移不变性 |
| 交叉熵损失 | L = -Σ y_i log(ŷ_i)；与 softmax 搭配求导得 ŷ - y（简洁梯度） | 给定标签和预测，计算 loss 值 |
| 矩阵求导 | dW = X^T dZ（线性层）；dX = dZ W^T；偏置 db = Σ dZ（按列求和） | 给定 X, W, Z 计算梯度形状 |
| BatchNorm | μ,σ² 为 mini-batch 均值方差；x̂ = (x-μ)/√(σ²+ε)；训练用 batch 统计量，推理用移动平均 | 判断 BN 层的参数个数 / 训练 vs 推理差异 |
| 激活函数导数 | sigmoid: σ(1-σ)；tanh: 1-tanh²；ReLU: 0 或 1 | 计算梯度值 / 判断梯度消失问题 |
| 参数量计算 | Linear(n→m): nm+m；Conv(C_out, C_in, k): C_out×C_in×k² + C_out | 给定网络结构算参数量 |

## 五、易混淆速查

| 概念 | 区别要点 |
|------|----------|
| 不相关 vs 独立 | 不相关仅 Cov=0（线性）；独立 ⇒ 不相关，反之不一定（非线性关系） |
| 方差 vs 标准差 | Var = E[(X-μ)²]；SD = √Var；单位不同 |
| SVD vs 特征分解 | 特征分解要求方阵；SVD 任意矩阵可用；A^T A 的特征值 = SVD 奇异值的平方 |
| MLE vs MAP | MLE 无先验 → 过拟合倾向；MAP 加先验 → 等价正则化（L2 ↔ 高斯先验） |
| Jacobian vs Hessian | Jacobian 一阶（向量函数）；Hessian 二阶（标量函数）；Hessian 正定 → 凸 |
| Hessian 正定 vs 半正定 | 正定 → 严格极小；半正定 → 凸但可能平坦；不定 → 鞍点 |
| 行列式 vs 秩 | det ≠ 0 ⇔ 满秩 ⇔ 可逆；det=0 的几何意义是体积坍缩 |
| BN 训练 vs 推理 | 训练用当前 batch μ,σ²；推理用全局移动平均（running_mean, running_var） |
